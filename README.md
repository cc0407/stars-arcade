# stars-arcade
Arcade-style shooter, made using java.

## How to Use
- Install newest version of JRE
- Clone repo
- Run Stars-Arcade.jar from the eclipse-workspace_Stars directory
  - This jar file **MUST** be in this directory or else things will break
